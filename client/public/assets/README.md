# Print asset drop-zone

Place the following two files here so they show up on all Invoice, LPO and
Quotation prints when the **Sign & Stamp** toggle is on:

| File name        | Purpose                        | Recommended size |
| ---------------- | ------------------------------ | ---------------- |
| `signature.png`  | Sandeep — authorised signature | ~300×90 PNG, transparent background |
| `stamp.png`      | NXS company stamp              | ~180×180 PNG, transparent background |

If either file is missing, the print will fall back to a dashed placeholder box.
The toggle in the print dialog controls whether either image is rendered at all.

You uploaded these originals in the brief:

* `stamp-new-contracting.jpg`   → save here as **`stamp.png`**
* `SANDEEP-SIGN-NEW.jpg`        → save here as **`signature.png`**

(Prefer PNG with transparency; if JPGs are all you have, rename them to `.png`
extension — browsers still load them correctly for print.)
